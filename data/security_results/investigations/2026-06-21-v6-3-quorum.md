## Reflection — what the multi-frame structure produced beyond linear

A linear (single-attempt) probe would have likely:
- Picked ONE of these three frames and walked down it.
- Ratified the falsification on that single frame.
- Lost any cross-frame corroboration signal.

The three-frame + quorum structure produced additional value beyond the linear equivalent:

1. **Redundant corroboration on a load-bearing claim.** Frames 1 and 3, by different lens, both confirm that the repay fee is independent of reserve state. This is a structural redundancy — failure of one or the other argument would still leave the load-bearing claim supported by the surviving argument.

2. **Decoupled discovery across categories.** A linear probe sent down the repay-timing path might have missed the cumulative-rate ceiling angle entirely; a linear probe sent down that path might have missed the CPI guard. The disjoint-frame structure forces the orchestrator to allocate attention across three independent categories.

3. **A real audit-saturation dataset growth.** N=3 versus N=2 is a substantive bound extension. The audit-saturation framing is now bounded across three substrates, each with source-grounded falsifications.

4. **Honest-zero discipline preserved.** Each frame explicitly logged a kill criterion, an evaluation, and a reflection. None of the frames silently coerced a pass.

The structure is doing exactly what the Ultrafuzz post-evaluation found useful: a small number of attempts, run independently on the same scaffold, with explicit adjudication. This session's three-attempt structure brings N-attempt-diversity into the orchestrator's own operating model without requiring a CLI subprocess.
